# 0. 프로젝트 개요

긴 콘텐츠 소비 부담을 줄이기 위해:

* AI 요약
* 관심사 큐레이션
* 하이라이트 추출

기능 제공.

---

# 1. 문제정의

현재 콘텐츠앱은:

* 긴 콘텐츠 소비 피로 높음
* 탐색 난이도 높음
* 완독률 감소

문제 존재.

---

## 데이터 현황

| 항목      | 수치  |
| ------- | --- |
| 평균 완독률  | 21% |
| 평균 체류시간 | 3분  |
| 재방문율    | 18% |

---

# 2. 목표

* 콘텐츠 소비 부담 감소
* 완독률 증가
* 재방문 증가

---

# 3. 핵심 기능

## 3-1. AI 3줄 요약

긴 콘텐츠를:

* 핵심 요약
* 주요 키워드
* 핵심 인사이트

형태로 제공.

---

## 3-2. 하이라이트 추출

중요 문장을 자동 강조.

---

## 3-3. 관심사 큐레이션

사용자 관심 카테고리 기반 추천 제공.

---

# 4. 정책

| 항목       | 정책      |
| -------- | ------- |
| 최대 요약 길이 | 300자    |
| 지원 콘텐츠   | 텍스트 콘텐츠 |
| 지원 언어    | KR 우선   |

---

# 5. 상태값

| 상태         | 설명    |
| ---------- | ----- |
| GENERATED  | 생성 완료 |
| PROCESSING | 생성 중  |
| BLOCKED    | 생성 제한 |
| FAILED     | 실패    |

## 부적절 콘텐츠
