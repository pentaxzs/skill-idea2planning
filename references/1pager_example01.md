# [26.05.17] AI 기반 초개인화 큐레이션 피드 제안 (커머스플랫폼팀)

---

# 1. 프로젝트 한 줄 요약

본 프로젝트는 상품 탐색 피로도를 줄이고 구매 전환율을 향상시키기 위해
AI 기반 개인화 큐레이션 피드를 제공하는 프로젝트입니다.

---

# 2. 대상

| 대상     | 상황/특징                 |
| ------ | --------------------- |
| 고객     | 원하는 상품을 빠르게 찾고 싶은 사용자 |
| 신규 사용자 | 구매 이력이 적은 고객          |
| 셀러     | 상품 노출 확대를 원하는 판매자     |

---

# 3. 문제 정의

## 고객 문제

* 상품 수가 많아 탐색 피로 증가
* 검색 중심 UX로 인해 발견 경험 부족
* 관심 없는 상품 반복 노출

## 비즈니스 문제

* 홈 피드 이탈 증가
* 추천 상품 CTR 감소
* 구매전환율 정체

---

## 데이터 기반 현황

| 항목          | 현재 수치 | 인사이트         |
| ----------- | ----- | ------------ |
| 홈 피드 이탈률    | 62%   | 첫 5초 내 이탈 증가 |
| 추천 상품 CTR   | 2.3%  | 전년 대비 감소     |
| 검색 의존 비율    | 71%   | 탐색 기능 약함     |
| 신규 유저 구매전환율 | 1.1%  | 개인화 부족       |

---

## Root Cause

### Case 1.

추천 기준이 카테고리 중심

### Case 2.

실시간 행동 반영 속도 느림

### Case 3.

운영자 중심 편성 구조

---

# 4. 목표

| 항목        | 목표   |
| --------- | ---- |
| 추천 CTR    | +35% |
| 구매전환율     | +18% |
| 홈 체류시간    | +22% |
| 신규 유저 구매율 | +15% |

---

# 5. 현황 및 분석

## 고객 인터뷰

* “비슷한 상품만 계속 나온다”
* “검색하지 않으면 원하는 걸 찾기 어렵다”

## 경쟁사 분석

* 숏폼형 상품 추천 증가
* 관심사 기반 큐레이션 강화 추세

---

# 6. 해결방안

## Phase 1. AI 큐레이션 피드

### 제공 기능

* 행동 기반 추천
* 관심사 자동 태깅
* 시간대 기반 피드 변경

### 예상 효과

* 탐색 효율 증가
* 신규 상품 발견 확대

---

## Phase 2. 생성형 AI 추천

### 제공 기능

* “오늘의 스타일 추천”
* AI 상품 조합 제안
* 이벤트/시즌 기반 자동 큐레이션

---

# 7. 리스크

| 영향도    | 리스크      | 대응 방안       |
| ------ | -------- | ----------- |
| High   | 추천 품질 저하 | 품질 학습 모델 운영 |
| Medium | 과도한 개인화  | 탐색 탭 유지     |
| Low    | 운영 비용 증가 | 자동화 확대      |

---

# 8. 중요도 및 긴급도

## 중요도 : 상

* 홈 전환율 핵심 개선 과제

## 긴급도 : 상

* 경쟁 서비스의 추천 UX 고도화 가속
