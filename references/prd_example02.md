# [26.05.17] AI 추천 기반 홈 개인화 개편 (커머스앱팀)

> 작성일: 2026.05.17
> 작성자: Commerce Product Team
> 상태: Draft

---

# 0. 프로젝트 개요

커머스앱 홈을 AI 추천 기반으로 개편하여:

* 상품 탐색 효율 증가
* 구매 전환율 증가
* 리텐션 증가

를 목표로 한다.

---

# 1. 문제정의

현재 홈은:

* 운영배너 중심
* MD 수동 편성 중심
* 개인화 부족

상태이다.

---

## 문제 데이터

| 항목      | 수치   |
| ------- | ---- |
| 홈 CTR   | 4.2% |
| 홈→상세 전환 | 12%  |
| 구매전환율   | 2.1% |

---

# 2. 목표

## 핵심 목표

* 홈 구매전환율 개선
* 개인화 추천 강화
* 체류시간 증가

---

# 3. 핵심 기능

## 3-1. AI 추천 피드

추천 기준:

* 최근 조회
* 구매 이력
* 찜
* 유사 사용자 행동

---

## 3-2. 탐색형 카드

예시:

* 오늘 인기 급상승
* 최근 많이 본 브랜드
* 재구매 추천

---

## 3-3. Dynamic Ranking

실시간 클릭률 기반 카드 재정렬.

---

# 4. 추천 정책

| 항목        | 정책  |
| --------- | --- |
| 최대 노출 상품  | 20개 |
| 동일 브랜드 제한 | 3개  |
| 품절상품      | 제외  |

---

# 5. 상태값

| 상태       | 설명    |
| -------- | ----- |
| ACTIVE   | 추천 가능 |
| HIDDEN   | 노출 제외 |
| SOLD_OUT | 품절    |

---

# 6. 예외처리

## 추천 데이터 부족

신규 유저:

* 인기상품 fallback 제공

---

## 추천 실패

추천 API timeout 시:

* 운영배너 fallback

---

# 7. KPI

| 항목    | 목표   |
| ----- | ---- |
| 구매전환율 | +25% |
| CTR   | +18% |
| 체류시간  | +12% |

---

# 8. 데이터 로깅

| 이벤트             | 설명      |
| --------------- | ------- |
| feed_impression | 피드 노출   |
| product_click   | 상품 클릭   |
| add_cart        | 장바구니 추가 |

---

# 9. 운영 정책

* MD 수동 pin 가능
* 특정 브랜드 강제 노출 가능
* 시즌 프로모션 우선 노출 가능
